## About FinTrack
This is an accounting web application similar to SAP/1C

## Installation
```bash
  git clone https://github.com/FinTrackhq/FinTrack.git
  cd FinTrack
  composer i 
  npm i
  ren .env.example .env
  php artisan key:generate
  php artisan migrate --seed
  npm run serve
```
